## subroot

## Usage

```console
subroot is tool to find root domains from crt.sh using Organization names

Usage:
  ./subroot [flags]

Flags:
   -on, -org-names string[]  Org name to use
   -rl, -root-level int      Root level of domain (ex: 2 means *.domain.com) (default 2)
```
